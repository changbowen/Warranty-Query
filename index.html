Warranty Query

Warranty query tool that support batch operations using httpwebrequests.

Motivation

Warranty query tool that support batch operations using httpwebrequests. Currently supports Lenovo, Dell machines.

Screenshot

Language & Requirement

  • VB application written in VB.net
  • .Net framework 4.5 or above.

Features

  • Currently supported brands include Lenovo (US & CN) and Dell.
  • Support copy & paste from & to the form.
  • Returning calculated start date and end date of warranty as well as the raw response as string.

Note

  • Lenovo US and CN may return different warranty information.
  • It is not very hard to add support for other brands as long as the website accept web requests for warranty query.
